# YuperMods

A pixel-styled website for downloading and browsing Melon Sandbox mods.

Features

* 🟦 Blue pixel-art theme
* 🎮 Retro game-inspired design
* 📦 Embedded Google Drive mod library
* 🐙 GitHub profile button
* ▶️ YouTube channel button
* 📱 Mobile-friendly layout
* 🌐 GitHub Pages compatible

Installation

1. Download or clone this repository.
2. Ensure the main file is named index.html.
3. Upload the repository to GitHub.
4. Enable GitHub Pages:
    * Go to Settings
    * Open Pages
    * Select Deploy from a branch
    * Choose main
    * Save

Your website will be available at:

https://YOUR_USERNAME.github.io/YuperMods/

About

YuperMods is a community-driven hub for Melon Sandbox mods. The site provides quick access to mod downloads through a public Google Drive folder while maintaining a fun pixel-art style inspired by sandbox games.
